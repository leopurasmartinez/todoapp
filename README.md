
# TODO APP

This app allows you to have a todo list, adding, editing and deleting todo items.

## Installation

    
### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Run unit tests
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```

### Compiles and minifies for production
```
npm run build
```
